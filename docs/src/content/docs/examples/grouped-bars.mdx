## How it's built

As with multi-series lines, pass named `series` to `<Chart>` — then render `<BarGroup>` for
side-by-side bars. Swapping it for `<StackedBars>` stacks the same series instead (set
`stackY` on `<Chart>` so the y domain spans the stacked totals).

```tsx
import { Chart, BarGroup, XAxis, YAxis, Grid, Legend, Tooltip } from '@chartistry/react';

<Chart
  width={620}
  height={340}
  data={data}
  x={(d) => d.quarter}
  xScaleType="band"
  series={[
    { key: 'Plan', y: (d) => d.plan, color: '#94a3b8' },
    { key: 'Actual', y: (d) => d.actual, color: '#4f46e5' },
  ]}
>
  <Grid />
  <YAxis />
  <XAxis />
  <BarGroup radius={3} />
  <Tooltip />
  <Legend />
</Chart>;
```
