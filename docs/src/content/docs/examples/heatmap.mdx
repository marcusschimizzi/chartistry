A heatmap encodes a value as **color** across two categorical axes — here sales by region
(rows) and month (columns).

<HeatmapExample client:load />

## How it's built

`<Heatmap>` is self-contained: it reads the chart's **x** accessor and plot, builds its own
column and row band scales (uniform cells, independent of `bandPadding`/`orientation`), and
draws both axes' labels — so you don't add an `<XAxis>`.

```tsx
import { Chart, Heatmap } from '@chartistry/react';

<Chart width={620} height={280} data={data} x={(d) => d.month} xScaleType="band" margin={{ left: 64, bottom: 32 }}>
  <Heatmap y={(d) => d.region} value={(d) => d.sales} showValues padding={2} radius={3} />
</Chart>;
```

`value` drives a **sequential** color scale (a light→dark blue ramp by default; pass `colors`
to change it, `colorDomain` to fix the range). `showValues` prints each cell's value in an
auto-contrasting color so it stays legible on light and dark cells.
